# VEDL - Virtual Elite Dragons League

## Project Structure
- `src/components/` - All React components
- `src/context/AppContext.tsx` - Global state management
- `src/lib/` - Types, storage utilities
- `src/app/` - Next.js app router pages

## Commands
- `npm run dev` - Start dev server
- `npm run build` - Build for production
- `npm run lint` - Run linter

## Architecture
- Client-side SPA with Next.js App Router
- Local storage for data persistence (structured for easy backend swap)
- Framer Motion animations throughout
- Dark-only theme with neon green (#39FF14) accents
